### 🚀 Як запустити локально

#### Способ 1: Python сервер
```bash
cd d:/проекти/сайти/толік
npm run build          # Збудувати bundle
python -m http.server  # Запустити локальний сервер (порт 8000)
# Відкрити http://localhost:8000 в браузері
```

#### Способ 2: Node.js сервер
```bash
cd d:/проекти/сайти/толік
npm run build
node server.js         # Запустити на порту 8080
# Відкрити http://127.0.0.1:8080 в браузері
```

#### Способ 3: Webpack dev server (якщо потрібна гарячa перезавантаження)
```bash
cd d:/проекти/сайти/толік
npm start              # Запустить dev server з гарячою перезавантаженням
```

### 🔧 Конфігурація гри (можна змінювати в GameScene.js)

- `this.lives = 5` — кількість життів
- `this.spawnRate = 1` — об'єкти на секунду на старті
- `this.fallSpeed = 200` — швидкість падіння об'єктів
- Ймовірності спавну об'єктів: регулюються в `spawnFallingObject()`
- Бонус за монтажника: 50 очок
- Бонус за голуба: 10 очок

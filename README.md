# Петро, монтажник і його наставник Валера 🛠️🐦

Інтерактивна мікро-квест про Петра, монтажника, який боїться висоти, та його ментора Валеру (голуба).

## 🎮 Як грати

1. **Відкрити файл** — просто double-click на `index.html` або перетягніть в браузер
2. **Читати історію** — цитуйте стан Петра на лівій панелі
3. **Робити вибори** — кліцайте на кнопки вибору
4. **Слухати звуки** — увімкніть 🔊 для Web Audio ефектів
5. **Знаходити пасхалку** — кліцайте по касці Петра 3 рази!

## 🚀 Запуск локально

### Спосіб 1: Просто відкрити (найпростіший!)
```bash
# Windows: double-click на index.html
# Або перетягніть файл у браузер
```

### Спосіб 2: Python сервер (якщо Web Audio не працює)
```bash
cd "d:/проекти/сайти/толік"
python -m http.server 8080
# Відкрити http://localhost:8080
```

### Спосіб 3: npm
```bash
npm run serve
# Відкрити http://localhost:8080
```

## ⚡ Особливості

- ✅ **Повна статика** — один файл, нічого не встановлювати
- ✅ **Синтезовані звуки** — Web Audio API генерує всі звуки
- ✅ **Анімовані переходи** — сцена рухається з вашим страхом
- ✅ **Адаптивний дизайн** — працює на мобільних
- ✅ **Історія змінюється** — різні сцени за вибором
- ✅ **Секретна пасхалка** — 3 кліки по касці!

## 🎯 Сценарії

| Путь | Кінець |
|------|--------|
| Вгору + Героїзм | 🎉 **Почесний Птах** (135м, -страх) |
| Вгору + Обережність | 🌸 **Горизонтальний Геній** (1м, стабільно) |
| Вниз + Втеча | 🤝 **Ґрунтовий Майстер** (безпечно) |
| 3 кліки по касці | 🕊️ **Людина-Голуб** (таємна) |

## 🎨 Візуальна схема

- **Фон**: темний, від страху до спокою
- **Строки**: лукаво жовті та червоні (警告)
- **Аватар**: обличчя змінюється при лякливості
- **Перьїнки**: впадають на переможних сценах

## 🔊 Звуки (генеруються вживу)

- 🔇 Кліки та кулінарні ефекти
- 🕊️ Голуб воркує (Валєра)
- 🪛 Дриль гудить (робочий)
- 😱 Лякливий крик (страх)
- ✨ Фанфар (перемога)

## 📱 Мобільна адаптація

- Однаколонна макет на малих екранах (< 860px)
- Великі кнопки, читаємий шрифт
- Все працює без тачпада
